# Hexlet Chat
Проект: Чат(Slack)

## Ссылка на приложение

[Развернутое приложение на Render](https://frontend-project-12-iwmv.onrender.com/)

## Установка и запуск локально

### Установка

```bash
make install
```

### Запуск

```bash
make start
```

Приложение будет доступно по адресу `http://localhost:5001`
